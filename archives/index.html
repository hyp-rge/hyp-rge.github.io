Object

Objects are Collections of key-value pairs.

We use an object to encapsulate properties that are highly related.

A Function in an object is usually called a “Method”.

Create an object “type”

Factory Function: a function that returns a new object.

Constructor Function: a function that returns this (refers to the object it belongs to).

Note: The naming convention we use for constructor functions is different.

For variables and factory functions, we use Camel Notation.oneTwoThree()

To name constructor functions, We use Pascal Notation. OneTwoThree()
