Basics, Operations and Flow Control

Some slight differences with C++

  • Strict Equality(Type+Value) ===

    Lose Equality(Value) ==

  • for…in loop is used to iterate over the properties of an object.

    Example 2: Use Bracket Notation when we don’t know ahead of time what property we are going to access. obj.key ❌ :(

  • for…of loop is used to iterate over the elements in an array.

    If we just change the keyword in to of in the code above, we will get the error message “obj is not iterable”.

    Logical Operators with Non-Booleans (+ Short-circuiting)

    false || 1 -> 1

    false || 1 || 2 -> 1

    true && 'name' ->"name"

Object

Objects are Collections of key-value pairs.

We use an object to encapsulate properties that are highly related.

A Function in an object is usually called a “Method”.

Create an object “type”

Factory Function: a function that returns a new object.

Constructor Function: a function that returns this (refers to the object it belongs to).

Note: The naming convention we use for constructor functions is different.

For variables and factory functions, we use Camel Notation.oneTwoThree()

To name constructor functions, We use Pascal Notation. OneTwoThree()
